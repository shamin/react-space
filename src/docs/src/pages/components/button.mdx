## Button

---

<button>Hello Button</button>


### Usage
Install button component from npm
```bash
npm install --save @space-kit/button
```

Add button component to your file

```jsx
import React from 'react'
import { PrimaryButton } from '@space-kit/button'

const MyComponent = () => {
  return (
    <PrimaryButton>Hello Button</PrimaryButton>
  )
}

```
