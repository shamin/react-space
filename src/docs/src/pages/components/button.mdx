## Button

---

<Button>Hello</Button>

### Installation

Install button component from npm

```bash
npm install --save @space-kit/button
```

### Usage

Add button component to your file

```jsx
import React from "react"
import Button from "@space-kit/button"

const MyComponent = () => {
  return <Button>Hello Button</Button>
}
```
